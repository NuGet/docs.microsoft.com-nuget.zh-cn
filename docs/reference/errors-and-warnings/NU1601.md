# <a name="nuget-warning-nu1601"></a>NuGet 警告 NU1601

<pre>Dependency specified was 'PackageId' (>= 3.5.0) but ended up with 'PackageId' 4.0.0.</pre>

### <a name="issue"></a>问题
直接的项目依赖项是已升级到更高版本比指定的项目。

### <a name="solution"></a>解决方案
更新到适当版本的项目中的依赖项。
