# <a name="nuget-error-nu5041"></a>NuGet 错误 NU5041

<pre>The readme file 'readme.md' is empty.</pre>


### <a name="issue"></a>问题 

指定为包自述文件的文件为空。


### <a name="solution"></a>解决方案

填充自述文件的内容。
