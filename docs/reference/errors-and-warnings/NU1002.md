# <a name="nuget-error-nu1002"></a>NuGet 错误 NU1002

<pre>'CLEAR' cannot be used in conjunction with other values</pre>

### <a name="issue"></a>问题
无效的输入以及清除关键字的组合。

### <a name="solution"></a>解决方案
清除单独使用，并忽略所有其他输入。
