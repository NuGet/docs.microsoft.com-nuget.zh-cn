# <a name="nuget-error-nu5015"></a>NuGet 错误 NU5015
<pre>project.json cannot contain multiple Target Frameworks.</pre>

### <a name="issue"></a>问题

传递到 NuGet 包操作的 project.json 文件包含多个目标框架。


### <a name="solution"></a>解决方案

请修复将包含单个目标框架的 project.json 文件。
