# <a name="nuget-error-nu1001"></a>NuGet 错误 NU1001

<pre>The project 'Project' does not specify any target frameworks in 'ProjectFile'</pre>

### <a name="issue"></a>问题
项目不包含一个或多个框架。

### <a name="solution"></a>解决方案
添加`TargetFramework`或`TargetFrameworks`属性设置为指定的项目文件。
