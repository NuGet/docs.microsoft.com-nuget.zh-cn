# <a name="nuget-error-nu5038"></a>NuGet 错误 NU5038
<pre>The readme file 'readme.txt' has an invalid extension. It must end in .md.</pre>

### <a name="issue"></a>问题

自述文件必须是 markdown 文件，并且必须以 "*md*" 结尾。

### <a name="solution"></a>解决方案

对自述文件使用 markdown 文件扩展名。
