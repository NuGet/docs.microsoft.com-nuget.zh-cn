# <a name="nuget-error-nu5017"></a>NuGet 错误 NU5017
<pre>Cannot create a package that has no dependencies nor content.</pre>

### <a name="issue"></a>问题

正在保存的包不包含任何包引用、程序集或框架引用。


### <a name="solution"></a>解决方案

请确保正在创建的包包含程序集或包引用。
